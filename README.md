# Proyecto: Consumo de API - Jugadores

## Descripción
Esta aplicación web consume una API de deportes para mostrar información de jugadores de fútbol en forma de tarjetas. El proyecto tiene como objetivo practicar el uso de JavaScript para obtener datos externos y mostrarlos  en una página web.

## API utilizada
Utilize la API de TheSportsDB  
Link: https://www.thesportsdb.com/api.php

## Tecnologías utilizadas
- HTML
- CSS
- JavaScript

## Funcionalidades
- Buscar jugadores de fútbol mediante una API
- Mostrar información como imagen, nombre y equipo
- Limitar la cantidad de resultados mostrados

## Cómo ejecutar
1. Descargar  el proyecto
2. Abrir el archivo `index.html` en el navegador
3. Asegurarse de tener conexión a internet, ya que la aplicación consume una API externa
